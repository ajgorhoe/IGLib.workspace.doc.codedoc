
This directory contains style overrides.

doxygen_extrastyles.css - styles for generated  documentation that
  override the defaults. To include this style, point to the file
  with HTML_EXTRA_STYLESHEET directive in .dox files.
doxygen_teststyles.css - test styles for experimentation, can be
  applied to test.dox

Directory original/:
  customdoxygen.css: default style sheet used by Doxygen. Use as template for the overriding style sheet.
  header.html: Default header used by Doxygen, can be replaced 
    via HTML_HEADER configuration option.
  footer.html: Default footer, can be replaced via the HTML_FOOTER.
