
# The `images/` Directory of the `codedoc` reository

The IGLib logos are from the [IGLibCore repository](https://github.com/ajgorhoe/IGLib.modules.IGLibCore/).

The C# logos (`Logo_C_sharp.svg` and `Logo_C_sharp_44x50.svg`) are from [this page on Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Logo_C_sharp.svg) and are in public domain.

The basic VTK logo `Visualization_Toolkit_logo.svg` was obtain from [this page on Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Visualization_Toolkit_logo.svg) and is in public domain. The other VTK logos were created for this one by text editing the SVG file (`IGLib_33x50.svg`) and by exporting to PNG format and resizing in IrfanView (`VisualizationToolkit_Logo_large.png`, `VisualizationToolkit_Logo_92x50.png`).
